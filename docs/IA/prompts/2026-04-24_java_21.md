# Prompt De Sesion - Java 21

## Prompt original

Quiero cambiar de java 17 a java 21.

## Objetivo interpretado

Actualizar la configuracion actual del proyecto Maven de RavenShop para compilar con Java 21 y alinear la documentacion operativa.

## Alcance acordado

- Cambiar `<java.version>` en `pom.xml`.
- Actualizar referencias activas del README de Java 17 a Java 21.
- Añadir en `AGENTS.md` que la version de Java del proyecto es Java 21.
- Registrar la sesion en la trazabilidad IA del proyecto.
- Verificar con Maven.
