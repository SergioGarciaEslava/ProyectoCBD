# Prompt De Sesion - Java 21

## Prompt recibido

Quiero usar java 21 en vez de java 17.

## Interpretacion

Actualizar la version Java del proyecto Maven de RavenShop de 17 a 21 y mantener la documentacion operativa alineada.

## Alcance aplicado

- Cambiar `<java.version>` en `pom.xml`.
- Actualizar referencias de requisitos y stack en `README.md`.
- Verificar compilacion y pruebas con Maven.
