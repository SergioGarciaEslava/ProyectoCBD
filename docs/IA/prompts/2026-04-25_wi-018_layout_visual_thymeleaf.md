# Prompt - WI-018 layout visual comun Thymeleaf

## Fecha

2026-04-25

## Herramienta

Codex con skill `ui-ux-pro-max`

## Prompt o resumen fiel

Implementar WI-018 para definir un layout visual comun en RavenShop con Thymeleaf y CSS propio.

El usuario pidio usar una paleta concreta:

- `#3d348b` indigo velvet
- `#7678ed` medium slate blue
- `#f7b801` amber flame
- `#f18701` tiger orange
- `#f35b04` cayenne red

Tambien pidio un estilo minimalista, sin abusar de cards y evitando una apariencia claramente generada por IA. La implementacion debia usar la skill UI/UX Pro Max como apoyo de diseno y mantenerse dentro de Spring MVC + Thymeleaf, sin frameworks frontend complejos.

## Restricciones aplicadas

- Sin React, Vue, Angular ni dependencias frontend nuevas.
- Sin autenticacion ni funcionalidad nueva de negocio.
- CSS global propio en `src/main/resources/static/css/app.css`.
- Aplicacion de estilo comun a home, productos, clientes y pedidos para evitar pantallas visualmente incoherentes.
