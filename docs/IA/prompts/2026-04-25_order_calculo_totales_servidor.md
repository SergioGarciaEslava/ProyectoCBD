# Prompt

El usuario pidio anadir logica de negocio simple para calcular importes de pedido en servidor dentro del proyecto academico RavenShop.

Requisitos:

- Calcular `lineTotal` de cada `OrderLineItem` a partir de `quantity` y `unitPrice`.
- Calcular `total` de `Order` como suma de `lineItems`.
- Mantener una implementacion pequena y defendible, preferiblemente en servicio o metodo de dominio sencillo.
- No crear formulario de pedidos.
- No introducir arquitectura compleja.
- No recalcular en frontend.
- Dejar claro que el total se calcula en servidor.
